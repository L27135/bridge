# Bridge iOS Prototype

This folder contains SwiftUI source sketches for the Bridge consumer and provider apps.

The UI is built from the shared product brief in `../../DESIGN_DNA.md`:

- consumer app is chat-first
- internal safety/mode logic is not displayed to the user
- profile building is quiet and reviewable
- consent controls what leaves the vault
- provider view is compact, source-labeled, and patient-approved

Full iOS compilation was not run in this workspace because `xcodebuild` reports that full Xcode is not selected; only Command Line Tools are active.
